Q1: 简述浏览器输入URL地址后发生的事情？

1.1 描述：
1.浏览器像DNS服务器发送URL获取对应的IP地址
2.DNS返回网站的IP地址
3.浏览器根据IP地址和目标服务器的80端口建立TCP连接
4.浏览器获取请求网站的html代码
5.浏览器在显示窗口渲染html内容
6.窗口关闭, 浏览器终止和服务器的连接
1.2 TCP知识点补充
